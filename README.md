## Проекты

### [GP2\_team\_Moscow\_cafe](https://github.com/afzalsgtv/GP2_team_Moscow_cafe) — групповой проект, ВШЭ 2025

Геоаналитика для выбора локации кафе в Москве. Город разбит на 1709 ячеек 800×800 м; по каждой собраны данные о трафике, конкуренции, аренде и инфраструктуре. На их основе рассчитывается интегральный индекс привлекательности — результат визуализирован в виде интерактивных тепловых карт.  
`Python` `GeoPandas` `OSMnx` `Folium` `Selenium` `pandas`

### [linear-models-and-knn](https://github.com/afzalsgtv/linear-models-and-knn) — учебный проект

Реализация и сравнение двух базовых семейств алгоритмов ML: kNN (включая собственную реализацию с нуля) и линейной регрессии с Ridge/Lasso-регуляризацией. Датасеты — Palmer Penguins и Diamonds. Подбор гиперпараметров через кросс-валидацию, диагностика мультиколлинеарности, отбор признаков через Lasso.  
`Python` `scikit-learn` `NumPy` `pandas` `matplotlib`
